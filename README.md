# Azure SIEM Lab with Microsoft Sentinel

## Status
Planned

## Project Overview
This project will document the process of building a basic cloud-based SIEM lab using Microsoft Azure and Microsoft Sentinel.

## Project Goals
- Create a Microsoft Sentinel workspace
- Collect security logs from a Windows virtual machine
- Use KQL to query security events
- Identify suspicious login activity
- Create a basic detection rule
- Document findings with screenshots

## Tools Planned
- Microsoft Azure
- Microsoft Sentinel
- Log Analytics Workspace
- Windows Virtual Machine
- KQL

## Planned Deliverables
- Setup steps
- Screenshots
- KQL queries
- Detection rule
- Findings
- Lessons learned
- Cleanup notes

## Notes
This project is currently planned and will be updated as the lab is built.
